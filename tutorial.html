Hello, world

In this tutorial we'll build a simple "Hello, world" module to show and explain each 's feature.

Loading in host page

First of all we need a page to include . So, create a blank html file. You don't need to download if you don't want to. You can link to CDN: 

	<script src="http://cdn.zentric.es/flink/beta/dist"><script>
</body>

Creating module

The next step is create the module file which contains our code. To make things as easy as possible, provides a BLANK module template to start with. Download and rename as module.Hello.js, then edit and do next:

Set moduleKEY.
Find the line that sets the var named moduleKEY and assign the value HelloWorld. This is the module identifier which will be registered as Flink.HelloWorld, so it must be unique along the namespace. 
var moduleKEY = "HelloWorld";
Code the call method.
Find the function "call" and add some awesome and original executable code: 
function call(method, args)
{
	alert("Hellow, world");
}

That's all we need. Now we have an operative module ready to work.
